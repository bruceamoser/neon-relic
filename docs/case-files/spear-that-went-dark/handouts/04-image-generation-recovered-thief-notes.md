# IMAGE GENERATION FILE — Recovered Thief Notes

Use this file to generate the final image for *Planned Plate D: Recovered Thief Notes*.

## Target output

- final asset path: `assets/png/spear-handout-recovered-thief-notes.png`
- orientation: portrait notebook-page or loose-sheet image
- use case: in-book reproduction of the thief's notes as physical evidence

## Core visual goal

The image should look like a real page of panicked field notes from a criminal participant whose understanding is collapsing. It should feel hurried, damaged, and human, but still readable enough to function as a handout.

## Required visual elements

- wrinkled notebook paper, torn pad sheet, or rough loose page
- ballpoint handwriting or mixed writing styles suggesting stress and interruption
- crossings-out, pressure marks, uneven spacing, or hurried corrections
- subtle field damage: fold lines, thumb grime, edge wear, optional faint smudging

## Tone guidance

- fear and confusion, not occult calligraphy
- believable criminal note-taking under pressure
- should imply a breakdown in the planned sale without turning into theatrical madness

## Period constraints

- late-1980s found-document realism only
- no modern notebook branding, no gel pens, no crisp designer stationery
- no excessive gore, no horror-movie blood effects, no supernatural burn marks

## Recommended master prompt

`Recovered criminal field notes from 1987 Buenos Aires, handwritten on wrinkled notebook paper or a torn loose sheet, frantic ballpoint writing, crossed-out phrases, pressure marks, fold creases, realistic analog found-document photography, anxious and human rather than theatrical, minimal restrained smudging, no modern paper branding, suitable for a printed RPG evidence handout`

## Negative prompt / exclusions

- no fantasy parchment
- no occult symbols unless incidental and realistic
- no blood-soaked shock image
- no cinematic prop exaggeration

## Acceptance check

The page should read as evidence, not as artwork about evidence. If it looks staged for a horror poster, regenerate.
