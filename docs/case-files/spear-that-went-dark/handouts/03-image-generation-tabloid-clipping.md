# IMAGE GENERATION FILE — Tabloid Clipping

Use this file to generate the final image for *Planned Plate C: Tabloid Clipping*.

## Target output

- final asset path: `assets/png/spear-handout-tabloid-clipping.png`
- orientation: portrait or slightly cropped news-clipping plate
- use case: player-facing reproduced tabloid evidence in the sample case chapter

## Core visual goal

Generate a believable late-1980s sensational newspaper clipping about the Recoleta robbery aftermath. It should look cheap, urgent, and commercially exploitative rather than respectable. The clipping should imply blackout reports, bleeding witnesses, and public unease without tipping into supernatural fantasy design.

## Required visual elements

- loud tabloid headline treatment
- cheap newsprint texture
- halftone photo box or low-quality inset image area
- clipped or torn paper edges acceptable if restrained
- text columns that feel like real tabloid layout rather than a poster

## Tone guidance

- sensational but not camp parody
- should feel like an actual 1987 paper trying to sell panic and coincidence
- public-facing language, not institutional language

## Period constraints

- late-1980s print production only
- no full-color modern tabloid design
- no desktop-publishing cleanliness
- no contemporary photojournalism sharpness

## Recommended master prompt

`1987 tabloid newspaper clipping about blackout trails, robbery aftermath, bleeding witnesses, and religious panic in Buenos Aires, cheap newsprint, bold sensational headline, dense text columns, halftone photo box, aged clipping edges, analog print imperfections, realistic late-1980s tabloid layout, no modern design polish, no supernatural glow, suitable for a tabletop RPG handout plate`

## Negative prompt / exclusions

- no modern newspaper masthead styling
- no glossy magazine color treatment
- no meme-like exaggeration or parody layout
- no horror-poster composition

## Acceptance check

The finished image should look like a clipping a DA could hand across the table without explanation. If it looks like cover art instead of disposable print media, regenerate.
