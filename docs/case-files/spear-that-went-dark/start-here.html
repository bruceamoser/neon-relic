Neon Relic — Sample Case File
Case: The Spear That Went Dark • VC-AR-87-041 • Buenos Aires, Argentina

This folder contains everything you need to run The Spear That Went Dark — a complete sample case file for Neon Relic. The documents are organized into the 8-piece case file package described in Chapter 15. Print them, spread them on the table, and you are ready to play.

If this is your first case, read the Case Brief — DA first. It tells you what is actually happening, who is really involved, and how the case resolves. Then hand the Agent Briefing to the players — it contains only what the Covenant would share with field agents.

Player Handout

Give this to the players at the start of the session.

  • 1. case-brief-player.html — The Agent Briefing. Situation summary, objectives, known organizations, starting contacts, and operational parameters. No spoilers.

DA Documents

These are for your eyes only. Do not share them with the players.

  • 2. case-brief-da.html — The DA’s master reference. Mystery statement, the real situation, containment truths, relic milestones, key actors, resolution paths, and pacing notes. Read this first.
  • 3. relic-sheet.html — Full artifact profile for the Spear of Destiny. Tier, category, activation condition, mechanical effects, fracture table, and containment truth checklist.
  • 4. operations-board.html — The Operations Board. Shift row with relic milestones, six organization countdown rows, and packet tracking. Print this and put it where the table can see it.
  • 5. organization-reference.html — All six organizations with their O#M# milestone descriptions and cross-link consequences. Keep this behind the screen.
  • 6. locations.html — Seven location pages (L1–L7). Pull the relevant page when the players say where they’re going.
  • 7. npc-cards.html — Eight NPC Cards. Place the relevant cards next to the active location page during play.
  • 8. information-cards.html — Twenty-two Information Cards (I1–I22). Player-facing front, DA reference back. Hand players the front side when they discover information.

Handouts

The handouts/ subdirectory contains all the visual assets for the case — clue handout images, NPC portraits, location images, and the relic illustration. Print or display these during play as physical props for the table.

Running order: Read the Case Brief. Skim the Operations Board and Organization Reference to understand the pressure structure. Glance through the Location pages and NPC Cards so you know the cast. Then sit down and play. You do not need to memorize anything — the forms are designed to be consulted during the session.

Building Your Own

Once you have played or read through this case, the step-by-step guide in Building a Case File (included in the rulebook) walks you through creating your own case from scratch, referencing the blank template files in the blank-templates/ folder.
