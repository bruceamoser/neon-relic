# Shift Timeline Handout

Use this handout at the table to track ten in-fiction days at four shifts per day.

## Shift Key

- `Morning`
- `Day`
- `Evening`
- `Night`

## How to use

Write short notes directly in each block:

- packet sent
- packet due
- library opens
- police shift change
- stakeout payoff
- witness transfer
- Church office available
- likely cargo window
- front sign appears
- relic event hits
